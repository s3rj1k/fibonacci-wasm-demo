Fibonacci Sequence Calculator

Enter a number to get the first n Fibonacci numbers starting from 0.

Examples
Valid Examples:
  • n = 5 → [0, 1, 1, 2, 3]
  • n = 8 → [0, 1, 1, 2, 3, 5, 8, 13]
  • n = 0 → []
  • n = 1 → [0]
Error Cases:
  • n < 0 Negative numbers
  • text Non-numeric input
  • empty No input provided
